# Wordfeud wordlists
A collection of some wordlists that can be used in wordfeud.

## Dutch wordlist
In directory `nl`:

`wordlist-nl.txt`:  based on the [OpenTaal word list](https://www.opentaal.org).

It has been processed by the author.

## Other wordlists

Here are some links to other wordlists. They are not maintained by me, and may need some processing before they are can be used.
Specifically:
* remove any comment lines
* remove words that can not be formed by the tile set for that language
* remove words that use more than 15 tiles (improves efficiency)

Links:

* [Wordfeud dictionaries](https://wordfeud.com/dictionaries/)

* [Wordfeud Languages](https://www.word-grabber.com/word-lists/wordfeud-languages)
